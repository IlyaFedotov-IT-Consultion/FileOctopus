# FileOctopus MVP Usage

## Keyboard Shortcuts

| Shortcut               | Action                                         |
| ---------------------- | ---------------------------------------------- |
| `Tab`                  | Switch active panel                            |
| `Enter`                | Open selected folder or file                   |
| `F2`                   | Rename selected item                           |
| `Delete`               | Open Trash confirmation for selected items     |
| `Shift+Delete`         | Open permanent delete confirmation             |
| `Ctrl/Cmd+C`           | Copy selected items to the internal clipboard  |
| `Ctrl/Cmd+X`           | Cut selected items to the internal clipboard   |
| `Ctrl/Cmd+V`           | Paste internal clipboard into the current pane |
| `Ctrl/Cmd+A`           | Select all visible items                       |
| `Backspace` / `Alt+Up` | Navigate to parent folder                      |
| `Alt+Left`             | Navigate back                                  |
| `Alt+Right`            | Navigate forward                               |
| `F5` / `Ctrl/Cmd+R`    | Refresh active panel                           |
| `Ctrl/Cmd+L`           | Focus editable path entry                      |
| `Escape`               | Close active dialog or cancel path editing     |

Shortcuts are ignored while typing in text fields.

## MVP Core File Manager Baseline

The desktop MVP supports sidebar standard locations, breadcrumb and raw path navigation, OS default file opening, reveal in file manager, details/list/icon view modes, sorting, current-folder filtering, recursive filename search, hidden-file toggling, context menus, properties, create folder, create empty file, copy, cut, paste, rename, Trash, and confirmed permanent delete.

## Diagnostics

The Diagnostics panel shows app version, build profile, target OS, commit SHA when available, schema version, recovered interrupted job count, data directory, and log directory. Use the export action to create a bug-report bundle.
